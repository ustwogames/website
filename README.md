## Dependencies

* [Node (4.2+)](https://nodejs.org/)
* [NPM (3.0+)](https://www.npmjs.com/)

## Usage

### Install NPM dependencies

```sh
npm install
```

### Run development server

```sh
npm start
```

### Build for production

```sh
npm run build
```
